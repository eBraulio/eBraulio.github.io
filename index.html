Find your Favorite Artist's Discography (Spotify API)

Project Overview

The site is a frontend application developed for the TripleTen Full-stack WEB Developer bootcamp. Built with React + Vite, it connects to the Spotify API and is deployed using Firebase with Google Sign-In authentication. The project highlights frontend development, API integration, and deployment skills.

Features

  • Frontend: Developed with React + Vite for a modern, fast interface.
  • Deployment: Hosted on Firebase for scalability and reliability.
  • JAuthentication: Google Sign-In with session persistence for seamless user experience.
  • Spotify API: Fetches artist details and albums, showcasing API integration.
  • Security: Authentication credentials stored securely in a .env file.

Key Achievements

  • Built and deployed a functional frontend using React + Vite and Firebase.
  • Integrated Google Sign-In with session persistence for improved UX.
  • Connected to the Spotify API to fetch and display artist and album data.
  • Secured sensitive data using environment variables.
  • Shared the project on GitHub for collaboration and feedback.

Summary

This project demonstrates my ability to build, deploy, and integrate modern web applications. Using React + Vite, Firebase, and the Spotify API, I created a user-friendly app with secure authentication and seamless data fetching.

Project Overview

This project is a full-stack web application hosted on Google Cloud and linked to a FreeDNS domain. It includes both frontend and backend deployment for the "Alrededor de los EE.UU." project. The frontend, built with React, handles user registration, authentication, and token management, while the backend, powered by Express.js and MongoDB, provides a RESTful API for user and card data. The application is secured with SSL and utilizes modern development practices.

Features

  • Frontend: User registration and authentication. React components for Login, Register, ProtectedRoute, and InfoToolTip. Local storage and token manipulation.
  • Backend: MongoDB for NoSQL database management. Schemas and models for Users and Cards. Error handling with status codes (400, 404, 500).
  • Deployment: Hosted on Google Cloud with FreeDNS domain. Secured with SSL and managed using Nginx and pm2.

Key Achievements

  • Developed a full-stack application with React frontend and Express.js backend.
  • Implemented user authentication, token management, and secure routing.
  • Deployed the application on Google Cloud with SSL for secure access.
  • Utilized FreeDNS for domain management and Nginx for server optimization.

Summary

This project showcases my ability to build and deploy a full-stack web application. Combining React for the frontend and Express.js with MongoDB for the backend, I created a secure and scalable platform for user interaction. Hosted on Google Cloud with SSL and managed via Nginx, the application reflects my expertise in modern web development and deployment practices.
